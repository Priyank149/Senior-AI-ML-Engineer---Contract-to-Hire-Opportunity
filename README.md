Assumptions: Driver is based in Dallas, TX with a home base in San Antonio, TX. Driver operates a Hotshot/Gooseneck trailer with a capacity of 14,000 lbs and requires a minimum effective rate of $2.00 per mile.

Profile Extraction: Driver details were extracted from the provided profile information, including location, equipment type, weight capacity, and minimum rate requirements.

Incomplete Data Handling: Loads with missing destination or price information were excluded because effective rate per mile could not be calculated reliably.

Rejected Load Example: A higher-paying load was rejected because it either did not match the driver's trailer type, exceeded capacity limits, or lacked sufficient information to calculate the effective rate per mile.
